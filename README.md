# ArchiScrapper

Foundation-first repository for a .NET 8 library ecosystem.

## Current Stage
This repository currently contains only the engineering foundation:
- project metadata and build standards,
- AI-assisted development instructions,
- working documentation,
- repository organization for future multi-project growth.

No business domain logic is implemented at this stage.

## Repository Structure
- `.github/` AI instructions and reusable prompts.
- `docs/` architecture and planning documentation.
- `src/` production projects (to be added).
- `tests/` automated tests (to be added).
- `samples/` usage examples (to be added).

## Engineering Baseline
- .NET 8 conventions via `Directory.Build.props`.
- Centralized NuGet versioning via `Directory.Packages.props`.
- Unified formatting and style rules via `.editorconfig`.
- Standard ignore rules via `.gitignore`.

## Next Step
Add the first production project under `src/` and matching tests under `tests/` while preserving these baseline standards.
