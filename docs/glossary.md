# Glossary

- Foundation: Baseline repository setup, standards, and workflow assets.
- Multi-project repository: A repository containing multiple related projects with shared conventions.
- Central Package Management: Single place to manage NuGet package versions.
- Quality gate: Automated rule that must pass (build, test, analysis) before merge.
- Decision log: Structured record of architectural decisions and rationale.
- Technical backlog: Prioritized list of engineering tasks and improvements.
